# 🚗 Smart Parking Management System

## 📌 Features
- Real-time slot booking
- QR code generation
- Exit management system
- Dashboard analytics
- Email notification
- User authentication (Login/Logout)

## 🛠️ Tech Stack
- Django (Backend)
- HTML, CSS, Bootstrap (Frontend)
- SQLite (Database)

## 🚀 How to Run
1. Clone the repo
2. Run `python manage.py runserver`
3. Open http://127.0.0.1:8000/
